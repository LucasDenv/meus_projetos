Características das caixas

  • Aninhamento: Uma caixa(Elemento) dentro de outra caixa(Elemento)(Aninhamento em forma de ninho uma coisa que cabe outra dentro que cabe outra dentro.).
  • Todo elemento visivel de um site é uma caixa.(Pode ser maior ou menor que o elemento.)

    • Medidas

    1. height: Atura.

    2. width: Largura

    3. border: Uma linha que circunda o seu conteúdo.

      1. padding: (Espaço interno) Acochoamento/preenchimento(Afasta a borda borda)
      2. margin: (Espaço externo) Borda
      3. outline: (tracejado dentro do margin) Contorno/Traçado


Tipos de Caixas

  1. Box-level: Sempre se inicia em uma linha nova. Ele sempre ocupa a largura total da tela. Depois pula pra proxima linha e continua.

    • div = divisões
    • header = Cabeçalho.
    • main = Conteúdo principal.
    • footer = rodapé
    • article = artigo
    • aside = conteúdo relatico ao artigo.
    • section = seção genérica, geralmente com um título.

  2. inline-level: Se inicia ao lado. Desenha a caixa do tamanho necessario pro conteudo depois continua ao lado.

    3. Ex: span
