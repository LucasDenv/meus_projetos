Seletores Personalizados

Se usa para especificar uma tag, para facilitar na hora de usar o CSS.

id

id = Só pode um elemento. Em CSS são representadas por "#".

class

class = Pode vários elementos. Em CSS são representadas por ".".

Pseudo-Classes e Pseudo-Elementos

Pseudo-Classes

Definem estados de elementos dentro do HTML. Tem que estar relacionadas a um elemento ou a uma classe. Em CSS sao representadas por ":".

    EX:
  • hover = muda quando passa o mouse.
  • visited = link visitado.
  • active = cor dele ativo.

Pseudo-Elementos

Pode agir no class, id, elementos, mexem diretamente no contrúdo periférico do elemento. Em CSS são representadas por " :: ".

    EX
  • after = depois do elemento.
  • before = antes do elemento.
  • .especial::after {...} = Só os links da class especial vão ter isso antes.
