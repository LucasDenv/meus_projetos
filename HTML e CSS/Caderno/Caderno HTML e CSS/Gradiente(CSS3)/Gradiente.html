Gradiente

 

Use a tag background-image: linear-gradient(90deg,cor,cor). Depois diga a direção do Gradiente e as cores.

  

*{height: 100%;}
/*Configurações globais das CSS*/

<Style>
        body{
            font-family: Arial, Helvetica, sans-serif;
            background-image: linear-gradient(90deg,cor,cor);
            color: black;
        }
</Style>
