    Aúdio

     
  • Tag: Audio..... Source:src...

    • Atributos da Tag.

  • Preload: Indica se áudio será pré-carregado ou não.

    • metadata: Vai carregar apenas as informações(Tamanho) tempo, informações.

    • none: Não vai carregar absolutamente nada até alguem ou script inicie.

    • auto: Vai carregar o arquivo de aúdio inteiro ao abrir a página.

  • Controls: Vai apresentar o play na tela.

  • Autoplay: Reproduz assim que a página se abre.

  • loop: loop.

  •                         
                            < audio preload="auto" autoplay controls>
                                < source src="audio.mp3" type="">
                                < source src="audio.ogg" type="">
                                < source src="audio.wav" type="">
                    
                            < /audio>
